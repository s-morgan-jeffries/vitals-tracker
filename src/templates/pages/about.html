About This App

This app consists of two parts: a front-end Backbone application and a back-end Node API server. Each was developed, built, and deployed separately using Grunt.

The front-end is a single-page Backbone application that's served from GitHub Pages. Most of the architecture was inspired/influenced by Pragmatic Backbone. Plots were generated using Rickshaw, an open-source plotting utility for time series data built on D3.js. CSS was generated using the Stylus port of Bootstrap.

The back-end is built with Strongloop's Loopback framework. It's deployed on Heroku. Data are persisted on a MongoDB instance hosted by MongoLab.
